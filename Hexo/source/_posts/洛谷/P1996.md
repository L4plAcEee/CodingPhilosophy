## Daily::算法 @[P1996 约瑟夫问题](https://www.luogu.com.cn/problem/P1996)
### 解
当你使用 `erase(it)` 删除当前元素之后，`it` 迭代器会失效，因此下一轮循环中 `it` 的初始位置会出错。你需要在 `erase` 之后立即更新 `it` 到下一个合法位置。
```cpp
int32_t main() {
    int n, m;
    cin >> n >> m;
    vector<int> a;

    for (int i = 1; i <= n; ++i) {
        a.push_back(i);
    }

    auto it = a.begin();
    while (!a.empty()) {
        for (int i = 1; i < m; ++i) {
            ++it;
            if (it == a.end()) it = a.begin();
        }
        cout << *it << ' ';
        it = a.erase(it);
        if (it == a.end()) it = a.begin();
    }

    return 0;
}
```
