## [P1449 后缀表达式](https://www.luogu.com.cn/problem/P1449)
### 思路
规则题目已经讲的很清晰了，难点在于某些边界情况。

### 解
```cpp
string expr;

int main() {
    cin >> expr;
    stack<int> stk;
    for (int i = 0, num = 0; i < expr.length(); ++i) {
        if (expr[i] == '@') break;
        if (isalnum(expr[i])) {
            num *= 10;
            num += expr[i] - '0';
        } else if (expr[i] == '.') {
            stk.push(num);
            num = 0;
        } else {
            int ans = 0;
            int op1 = stk.top();stk.pop();
            int op2 = stk.top();stk.pop();
            if (expr[i] == '+') {
                ans = op2 + op1;
            } else if (expr[i] == '-') {
                ans = op2 - op1;
            } else if (expr[i] == '*') {
                ans = op2 * op1;
            } else if (expr[i] == '/') {
                ans = op2 / op1;
            }
            stk.push(ans);
        }

    }
    cout << stk.top() << '\n';
}
```
